# nasa_explorer
NASA Explorer é um painel que conecta o usuário a três fontes da NASA: a foto astronômica do dia (APOD), asteroides próximos da Terra (NEO) e fotos dos rovers em Marte. O usuário navega, explora gráficos com dados reais, e mantém uma coleção pessoal de "Favoritos Cósmicos" (o CRUD).
